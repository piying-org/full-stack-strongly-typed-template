## nodejs全栈全链路强类型模板

- 数据库到后端的实体声明,类型及验证的强类型实现
  > piying-orm/typeorm
- 后端到前端的接口强类型实现
  > trpc
- 利用后端类型实现前端表单输入
  > piying-view/任意前端框架(Angular,Vue,React,Svelte,Solid)

## 特点
- 统一验证,只需要实现一次验证逻辑就可以同时在前/后端验证,提高了数据的安全性
- 类型的传递有迹可循,修改字段或属性后其他依赖这个类型的类型也会自动作出修改,降低了修改成本
- 开发体验好,虽然前后端项目分离,但是由于类型关联度高,所以开发时需要处理的前后端问题更少

## run

- npm i
- npm run init
- npm start
